## 💡 문제 요약

- N개의 집이 일렬로 있고 각 집을 빨강, 초록, 파랑 중 하나로 칠할 수 있음. 단, **연속으로 이웃한 두 집은 같은 색을 칠할 수 없음**
- 모든 집을 칠하는 데 드는 **최소 비용**을 구하라
- 집의 수 N(2 ≤ N ≤ 1000)

---

## 🧭 접근 방식

- 프로그래머스 땅따먹기 문제랑 같은 방법으로 풀면 되겠군!이라 생각해서 쉽게 풀었다.

---

## 💥 핵심 로직 & 코드

```jsx
for (let i = 1; i < N; i++) {
  for (let j = 0; j < 3; j++) {
    // 같은 열이 아닌 것만 체크해야 함.
    let min = Infinity;
    for (let k = 0; k < 3; k++) {
      if (j === k) continue;
      if (min > dp[i - 1][k]) min = dp[i - 1][k];
    }
    dp[i][j] += min;
  }
}
```
